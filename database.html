This database contains the hyperfine parameters of a number of different iron minerals typically investigated in Environmental sciences. To use the search function, simply insert a range of different values (i.e. minimum and maximum) and use the search buttons to refine the table. Currently you can only search values for one specific parameter at a time rather than a combination of different parameters.

Ox.
